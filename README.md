# BritaRH

O **BritaRH** é o sistema de recrutamento e seleção da **Britasul**, desenvolvido para facilitar o processo de candidatura a vagas da empresa. A plataforma permite que candidatos visualizem oportunidades disponíveis, realizem um teste inicial e enviem seu currículo de forma simples e rápida.

## 🚀 Funcionalidades

- **Visualização de vagas**
  - Lista com todas as vagas disponíveis na empresa.
  - Informações sobre cargo, requisitos e descrição da vaga.

- **Candidatura online**
  - O candidato pode se candidatar diretamente pelo sistema.

- **Teste inicial**
  - Questionário simples com **15 perguntas**.
  - Avalia conhecimentos básicos e perfil do candidato.

- **Envio de currículo**
  - Área para anexar o currículo após a realização do teste.
  - Suporte para arquivos em formato **PDF ou DOC/DOCX**.

## 📋 Fluxo de candidatura

1. O candidato acessa a plataforma **BritaRH**.
2. Visualiza as **vagas disponíveis**.
3. Seleciona a vaga desejada.
4. Realiza um **teste inicial com 15 perguntas**.
5. Após concluir o teste, é direcionado para a **página de envio do currículo**.
6. O currículo é enviado para análise do setor de **Recursos Humanos da Britasul**.

## 🏢 Sobre a Britasul

A **Britasul** utiliza o BritaRH para modernizar e organizar seu processo de recrutamento, tornando a seleção de talentos mais ágil, transparente e eficiente.

## 📌 Objetivo do sistema

- Facilitar a candidatura de novos profissionais.
- Automatizar etapas iniciais do processo seletivo.
- Centralizar o recebimento de currículos.
- Ajudar o RH a identificar candidatos com maior potencial.

## 🔐 Privacidade

Os dados enviados pelos candidatos são utilizados exclusivamente para fins de **recrutamento e seleção** da Britasul e tratados com confidencialidade.
